# Student Performance Analysis

This project demonstrates core Machine Learning concepts using a simple structured dataset.

## Concepts Covered
- Supervised Learning
- Regression
- Classification
- Unsupervised Learning
- Clustering
- Train/Test Split
- Feature Engineering

## Project Workflow
1. Data preparation
2. Feature and target definition
3. Train/Test split
4. Regression model implementation
5. Classification transformation
6. Clustering using KMeans

## Tools Used
- Python
- Pandas
- Scikit-learn

This project focuses on understanding the complete ML workflow from data preparation to evaluation.
